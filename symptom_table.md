# Symptom Progression Table

This table summarizes the reconstructed symptom patterns of the English Sweating Sickness based on primary sources, linguistic analysis, and mechanistic modeling.

| Phase | Symptom / Description | Notes |
|------|------------------------|-------|
| **Prodrome** | Sudden dread, fear, or foreboding | Reported as the earliest sign; autonomic surge indicator |
| | Inward heat / burning | Not fever; described as internal oppression or heat |
| | Headache or head heaviness | Often paired with dread |
| **Onset** | Abrupt collapse or weakness | Sudden loss of strength; hallmark feature |
| | Oppression of chest | Constriction, pressure, difficulty breathing |
| | Thirst | Intense, sometimes unquenchable |
| **Sweating Phase** | Profuse sweat | Continuous, heavy, sometimes foul-smelling |
| | “Stinking sweat” | Distinctive marker; suggests metabolic byproducts |
| | Restlessness or agitation | Patients urged not to move; movement worsened condition |
| **Neurological Signs** | Delirium or confusion | More common in fatal cases |
| | Tremors or shaking | Autonomic instability |
| **Fatal Progression** | Cold sweat | Indicates collapse of thermoregulation |
| | Labored breathing | Terminal respiratory failure |
| | Death within 3–18 hours | Consistent across outbreaks |
| **Survivor Pattern** | Prolonged sweating followed by resolution | Survivors often required 24+ hours of rest |
| | Relapse rare | Distinguishes from infectious fevers |

This table is intended as a structured reference for differential analysis and mechanistic modeling.
