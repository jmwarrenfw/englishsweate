Abstract
This repository presents a historically grounded, falsifiable hypothesis that the English Sweating Sickness (1485–1551) was not an infectious disease but a commodity‑driven toxicological event linked to specific trade‑network, class‑consumption, and material‑culture patterns in late medieval England. The model accounts for the syndrome’s demographic selectivity, abrupt onset, geographic confinement, micro‑temporal clustering, and final disappearance. The hypothesis is structured for evaluation by historical epidemiologists, geoanthropologists, and archaeotoxicologists.
